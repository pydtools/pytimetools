# [pytimetools](https://pypi.org/project/pytimetools/#description)
python datetime tools (py开发中的时间转换工具集合)

* utc
* gmt
* local time
* timestamp


local:
* local -> utc
* local -> timestamp
* local -> gmt

utc:
* utc -> local
* utc -> timestamp
* utc -> gmt

timestamp:
* timestamp -> utc
* timestamp -> local
* timestamp -> gmt

gmt:
* gmt -> utc
* gmt -> local
* gmt -> timestamp

## use in django 
* py3
* django(>=2.2)
* tz:settings.TIME_ZONE

由于目前依赖django, 所以使用py3

```shell
➜  ~ pip install pytimetools

```
